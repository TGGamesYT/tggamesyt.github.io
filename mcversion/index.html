MCVersion

MCVersion is a simple tool that runs in the background on your Windows PC. It watches for new Minecraft Java Edition releases and notifies you instantly with a toast popup when a new version comes out.

It works with a custom server and fetches version info regularly. When a new update is released, you’ll be notified and can click the popup to view the official article.

It runs on startup automatically and keeps your known versions up to date. Super lightweight and made for convenience!

⚠️ Warning:
When downloading and running the installer, your browser or Windows might say the file is unrecognized or unsafe. This is normal for unsigned software. The developer chose not to pay $70–$400 for a code signing certificate.

If you trust the source, click “More info” → “Run anyway” to proceed.
