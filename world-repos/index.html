World Repo Manager

Share and download Minecraft worlds through custom repositories

What is it?

World Repo Manager is a Minecraft mod that lets players browse and download Minecraft worlds directly from simple online repositories. Map creators can easily host their own worlds, and players can access them right from the game.

How does it work?

Create a MinecraftWorldRepo.json file and place it on your website, along with the zipped world files. Here‚Äôs a simple example: 

{
  "info": {
    "name": "My Test World Repo",
    "description": "Testing The mod",
    "author": "TGGamesYT"
  },
  "worlds": [
    {
      "name": "My test world",
      "description": "Testing!",
      "location": "test_world.zip"
    }
  ]
}

If your JSON is hosted at https://my.site/repo/MinecraftWorldRepo.json, the world zips must be placed at https://my.site/repo/test_world.zip.

How to use it?

Open Minecraft with the mod installed. From the main menu, click World Repo Manager, then click Add New Repo and paste the URL to your repo (e.g., https://my.site/repo). After that, you can browse and download any worlds listed in that repository.

Public Repositories

Want your repo listed in the public repo list shown in-game? Just open an issue on GitHub with your repo link and details: Submit Your Repo.
